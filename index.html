Graphite nodule statistics directly from SEM images

Run a released U-Net model in your browser or use the Python script for batch analysis. Uploaded images stay on your computer; the browser demo performs local ONNX inference.

Use the browser tool Download Python model
One public SEM example and reference overlay are provided. The full manuscript image set is not published in this repository.

Browser tool

Upload one representative low-magnification SEM field, enter the pixel size from your SEM scale bar, and run the model locally. Use the overlay as a quality check before using the metrics.

The browser tool is a quick single-image workflow. The Python script remains the recommended route for formal batch statistics and reproducible reporting.

Batch analysis

Use the Python script for multiple images, probability maps, masks, overlays, JSON summaries, and object-level CSV output.

git clone https://github.com/ustbTobyMa/adi-graphite-nodule-ai.git
cd adi-graphite-nodule-ai
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

python scripts/graphite_nodule_analyzer.py \
  --image-dir /path/to/sem_images \
  --checkpoint models/graphite_unet_seed7.pt \
  --output-dir /path/to/output \
  --pixel-size-um YOUR_MICROMETRES_PER_PIXEL
